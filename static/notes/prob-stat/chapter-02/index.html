第二章 离散型随机变量

2.1 随机变量的定义

随机变量

概率空间 (Ω,F,P) 是随机试验建模的基础，但很多问题关心的是随机试验中的定量特征，例如：

  • 随机挑选学生的身高；

  • 两个顾客到达商店的时间差；

  • 抽到某个稀有物品之前的抽取次数；

  • 传感器测量误差。

定义. 随机变量 X 是定义在样本空间上的实值函数

X:Ω→R,

并且满足：对任意区间 I⊆R，集合

{ω∈Ω∣X(ω)∈I}

是一个事件。

通常简记为

P(X∈I)=P({ω∈Ω∣X(ω)∈I}).

指示函数

给定事件 E，它的指示函数定义为

1E(ω)={1,ω∈E,0,ω∉E.

1E 是一个随机变量，并且

P(E)=E[1E].

2.2 离散型随机变量

定义与分布列

定义. 若随机变量 X 的值域是可数集，则称 X 为离散型随机变量。

它的分布列或概率质量函数定义为

pX(x)=P(X=x),x∈R.

分布列的基本性质

  • 非负性：

pX(x)≥0.

  • 只有至多可数多个 x 使 pX(x)>0。

  • 归一性：

∑x∈RpX(x)=1.

反之，若函数 p(x) 满足以上性质，则可以构造某个离散型随机变量，使它的分布列为 p(x)。

用分布列计算概率

对任意集合 B⊆R，有

P(X∈B)=∑x∈BpX(x),

其中求和只包括 pX(x)>0 的那些点。

离散型随机变量函数的分布

若 Y=g(X)，则 Y 仍为离散型随机变量，且

pY(y)=P(Y=y)=∑x:g(x)=ypX(x).

2.3 常见离散型分布

Bernoulli 分布

若 X 只取 0,1，且

P(X=1)=p,P(X=0)=1−p,

则称 X∼Bernoulli(p)。

事件指示函数 1E∼Bernoulli(P(E))。

二项分布

若

P(X=k)=(nk)pk(1−p)n−k,k=0,1,…,n,

则称 X∼Binomial(n,p)。

它描述 n 次独立 Bernoulli 试验中成功次数的分布。

几何分布

若

P(X=k)=(1−p)k−1p,k=1,2,…,

则称 X∼Geometric(p)。

它常用于刻画第一次成功所需等待的试验次数。

Poisson 分布

若

P(X=k)=e−λλkk!,k=0,1,2,…,

则称 X∼Poisson(λ)。

它常用于近似描述大量独立小概率事件在一段时间或空间区域中发生的次数。

离散均匀分布

若有限集 S 上每个元素等可能，即

P(X=x)=1|S|,x∈S,

则称 X 服从 S 上的离散均匀分布。

负二项分布

在 Bernoulli 过程中，令 Hr 表示取得第 r 次成功时的总试验次数，则

P(Hr=k)=(k−1r−1)pr(1−p)k−r,k=r,r+1,….

称 Hr 服从负二项分布。

2.4 离散型随机向量

联合分布列

若 X,Y 均为离散型随机变量，则 (X,Y) 为离散型随机向量。其联合分布列定义为

pX,Y(x,y)=P(X=x,Y=y).

联合分布列的基本性质

  • 非负性：

pX,Y(x,y)≥0.

  • 只有至多可数多个 (x,y) 使 pX,Y(x,y)>0。

  • 归一性：

∑x,ypX,Y(x,y)=1.

用联合分布列计算概率

对任意 B⊆R2，有

P((X,Y)∈B)=∑(x,y)∈BpX,Y(x,y).

边缘分布列

由联合分布列可以推出边缘分布列：

pX(x)=∑ypX,Y(x,y),
pY(y)=∑xpX,Y(x,y).

联合分布列能确定边缘分布列；反过来，边缘分布列一般不能唯一确定联合分布列。

2.5 随机变量的独立性

定义

随机变量 X1,…,Xn 若对任意区间 I1,…,In，都有

P(X1∈I1,…,Xn∈In)=∏i=1nP(Xi∈Ii),

则称它们相互独立。

离散型随机变量的独立性判别

设 X,Y 为离散型随机变量，则以下命题等价：

  • X,Y 相互独立；

  • 对所有 x,y，

pX,Y(x,y)=pX(x)pY(y);

  • 存在一元函数 g,h，使得

pX,Y(x,y)=g(x)h(y).

独立随机变量函数

若 X,Y,Z 相互独立，则 f(X,Y) 与 g(Z) 相互独立。

直观上，一个只由前两个变量决定的量，不会受到第三个独立变量的影响。

随机变量和的分布

令 Z=X+Y。若 X,Y 为离散型随机变量，则

pZ(z)=∑xpX,Y(x,z−x).

若进一步 X,Y 相互独立，则

pZ(z)=∑xpX(x)pY(z−x).

这就是离散情形下的卷积公式。

2.6 期望与方差

离散型随机变量的期望

若

∑x|x|pX(x)<∞,

则 X 的期望定义为

E[X]=∑xxpX(x).

若上述绝对收敛条件不满足，则称期望不存在。

LOTUS 公式

若 Y=g(X)，且右端绝对收敛，则

E[g(X)]=∑xg(x)pX(x).

若 (X,Y) 为离散型随机向量，h 为二元函数，则

E[h(X,Y)]=∑x,yh(x,y)pX,Y(x,y).

期望的基本性质

当相关期望存在时：

  • 线性：

E[aX+bY]=aE[X]+bE[Y].

  • 若 X≥0 几乎必然成立，则

E[X]≥0.

  • 若 X≥Y 几乎必然成立，则

E[X]≥E[Y].

  • 若 X1,…,Xn 相互独立且期望存在，则

E[∏i=1nXi]=∏i=1nE[Xi].

注意：期望的线性不要求独立性。

方差

若 E[X] 与 E[X2] 存在，则

Var⁡(X)=E[(X−E[X])2].

等价地，

Var⁡(X)=E[X2]−(E[X])2.

基本性质：

Var⁡(aX+b)=a2Var⁡(X).

若 X1,…,Xn 相互独立且方差存在，则

Var⁡(∑i=1nXi)=∑i=1nVar⁡(Xi).

2.7 Bernoulli 过程

定义

设 X1,X2,… 为一列随机变量， p∈(0,1]。若：

  • X1,X2,… 相互独立；

  • 每个 Xn∼Bernoulli(p)；

则称 (Xn) 为参数 p 的 Bernoulli 过程。

派生随机变量

  • Nn：前 n 次试验中成功的总次数。

Nn=∑i=1nXi.

于是

Nn∼Binomial(n,p).

  • Tk：第 k−1 次成功到第 k 次成功之间的等待时间。

每个 Tk∼Geometric(p)，且 T1,T2,… 相互独立。

  • Hr：取得第 r 次成功时的总试验次数。

Hr=T1+⋯+Tr.

于是 Hr 服从负二项分布。

几何分布的无记忆性

若 T∼Geometric(p)，则对 m,k≥1，

P(T−m=k∣T>m)=P(T=k).

直观上：已知前 m 次都失败之后，继续等待首次成功的分布与一开始相同。

Poisson 近似

给定 λ>0，令 pn=λ/n。当 n→∞ 时，

(nk)pnk(1−pn)n−k→e−λλkk!.

因此，当 n 很大、p 很小、np 适中时，可以用

Poisson(np)

近似 Binomial(n,p)。

本章小结

  • 随机变量是样本空间上的实值函数。

  • 离散型随机变量只有可数多个可能取值，由分布列完整刻画。

  • 联合分布列描述多个离散型随机变量之间的关系，边缘分布列由联合分布列求和得到。

  • 独立性在离散情形下等价于

pX,Y(x,y)=pX(x)pY(y).

  • 期望刻画平均水平，方差刻画波动大小。

  • Bernoulli 过程统一产生 Bernoulli、二项、几何、负二项等分布；Poisson 分布可以看成二项分布的小概率极限。
