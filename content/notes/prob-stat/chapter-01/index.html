第一章 概率论的基本概念

1.1 随机试验

随机试验与概率

  • 不确定现象：结果难以由已知信息和条件完全确定。
    • 可能来自物理机制过于复杂、内在不确定性、数据不足等。
    • 若某些不确定现象在大量重复后呈现稳定的统计规律，就可以用概率论建模。
  • 随机试验：可以在近似相同的条件下重复进行，并展现统计规律性的理想化过程。
    • 例：抛硬币、掷骰子、布朗运动、电子元器件热噪声。
  • 概率论：对随机试验进行数学建模的工具。

样本空间、事件与概率

  • 样本空间 ()：随机试验所有可能结果构成的非空集合。
    • 样本点通常记为 ()。
  • 事件 (E)：样本空间的子集。
    • 若本次试验结果 (E)，称事件 (E) 发生。
  • 概率 (P(E))：刻画事件 (E) 发生可能性大小的数。
    • 直观上，若重复试验 (n) 次，事件 (E) 的频率为 (_n(E))，希望

[ P(E)_{n}_n(E). ]

但现代概率论不直接把概率定义为频率极限，而采用公理化定义；频率收敛到概率是之后大数定律给出的结论。

1.2 集合论回顾

常用集合概念

  • (xA)：(x) 是集合 (A) 的元素。
  • (A=B)：当且仅当任意 (x) 都满足 (xAxB)。
  • (AB)：(A) 是 (B) 的子集。
  • (2^A)：集合 (A) 的幂集，即 (A) 的所有子集构成的集族。
  • 常见集合：
    • (={0,1,2,})
    • ()：整数集
    • ()：有理数集
    • ()：实数集

可数集与不可数集

  • 可数无限集：元素可以被一个无限长序列无遗漏地枚举出来。

    • 例：(,,)。

  • 可数集：有限集或可数无限集。

  • 不可数集：不是可数集的集合。

    • 例：() 及其实区间。

  • 常用结论：

    • 可数集的子集仍然可数。
    • 可数个可数集的并仍然可数。
    • 两个可数集的笛卡尔积仍然可数。

集合运算

  • 并集：

[ AB={xxAxB}. ]

  • 交集：

[ AB={xxAxB}. ]

  • 差集：

[ AB={xAxB}. ]

  • 补集：若全集由上下文给出，(B^c) 表示 (B) 的补集。

  • 德摩根律：

[ ({iI}A_i)^c={iI}A_i^c, ({iI}A_i)^c={iI}A_i^c. ]

  • 笛卡尔积：

[ AB={(a,b)aA, bB}. ]

1.3 样本空间、事件与概率空间

事件的关系与运算

  • (EF)：事件 (E) 发生时，事件 (F) 必定发生。
  • (E^c)：(E) 不发生。
  • (EF)：(E) 和 (F) 都发生。
  • (EF)：(E) 或 (F) 发生。
  • 若 (EF=)，称 (E) 与 (F) 互斥或互不相容。

事件域

并非所有情况下都能给样本空间的每个子集赋予合理概率，尤其当 () 不可数时。因此需要指定一个事件集族。

定义. 设 () 为样本空间，(^)。若 () 满足：

  • ()；
  • 若 (E)，则 (E^c)；
  • 若 (E_1,E_2,)，则

[ _{i=1}^{}E_i, ]

则称 () 为 () 上的一个事件域。

概率测度与概率空间

定义. 设 () 为 () 上的事件域。映射 (P:) 若满足：

  • 非负性：对任意 (E)，有 (0P(E))；
  • 规范性：(P()=1)；
  • 可数可加性：若 (E_1,E_2,) 两两互斥，则

[ P(_{i=1}{}E_i)=_{i=1}{}P(E_i), ]

则称 (P) 为 () 上的概率测度。

概率空间是三元组

[ (,,P), ]

其中 () 是事件域，(P) 是概率测度。

1.4 概率的基本性质

基本结论

  • (P()=1)。
  • (P()=0)。
  • 若 (E_1,,E_n) 两两互斥，则

[ P(_{i=1}{n}E_i)=_{i=1}{n}P(E_i). ]

  • 补集公式：

[ P(E^c)=1-P(E). ]

  • 若 (EF)，则

[ P(E)P(F). ]

  • 差集公式：

[ P(FE)=P(F)-P(E),EF. ]

  • 两事件并的概率：

[ P(EF)=P(E)+P(F)-P(EF). ]

几乎必然与零概率事件

  • 概率为 (1) 的事件不一定是必然事件，称为几乎必然事件。
  • 概率为 (0) 的事件不一定是不可能事件。

Boole 不等式

对任意事件 (E_1,,E_n)，有

[ P({i=1}^{n}E_i){i=1}^{n}P(E_i). ]

更一般地，对可数多个事件也有

[ P({i=1}^{}E_i){i=1}^{}P(E_i). ]

1.5 古典概型

定义

若概率空间 ((,,P)) 满足：

  • () 是有限集；
  • (=2^)；
  • 每个样本点等可能，即

[ P({})=,, ]

则称其为古典概型。

此时任意事件 (E) 的概率为

[ P(E)=. ]

计算思路

  • 先明确样本空间 ()。
  • 判断是否等可能。
  • 计算有利结果数 (|E|) 与总结果数 (||)。
  • 常用计数方法：
    • 加法原理；
    • 乘法原理；
    • 排列组合；
    • 补事件法。

1.6 条件概率

条件概率的定义

给定概率空间 ((,,P))。若事件 (F) 满足 (P(F)>0)，则事件 (F) 发生条件下事件 (E) 的条件概率定义为

[ P(EF)=. ]

若 (P(F)=0)，则 (P(EF)) 在这个定义下没有意义。

条件概率仍是概率测度

固定满足 (P(F)>0) 的事件 (F)，令

[ P_F(E)=P(EF). ]

则 (P_F) 是 () 上的一个概率测度。也就是说，在已知 (F) 发生之后，条件概率给出了更新后的概率空间。

古典概型中的条件概率

若原概率空间为古典概型，且 (F)，则

[ P(EF)=. ]

直观上：已知 (F) 发生后，只在 (F) 内重新等可能地计数。

乘法公式

若 (P(E_1E_{n-1})>0)，则

[ P(E_1E_n) =P(E_1)P(E_2E_1)P(E_nE_1E_{n-1}). ]

全概率公式

若 (F_1,,F_n) 构成样本空间的一个划分，即

[ F_1F_n=,F_iF_j= (ij), ]

且 (P(F_i)>0)，则对任意事件 (E)，

[ P(E)=_{i=1}^{n}P(EF_i)P(F_i). ]

Bayes 定理

在全概率公式的条件下，若 (P(E)>0)，则

[ P(F_iE)= {_{j=1}^{n}P(EF_j)P(F_j)}. ]

  • (P(F_i))：先验概率。
  • (P(F_iE))：观察到 (E) 后的后验概率。

1.7 事件的独立性

两个事件的独立性

定义. 若事件 (E,F) 满足

[ P(EF)=P(E)P(F), ]

则称 (E) 与 (F) 相互独立。

若 (P(F)>0)，则

[ EFP(EF)=P(E). ]

也就是说，已知 (F) 发生不会改变 (E) 的概率。

独立性的补事件性质

对任意事件 (E,F)，以下命题等价：

  • (E) 与 (F) 独立；
  • (E) 与 (F^c) 独立；
  • (E^c) 与 (F) 独立；
  • (E^c) 与 (F^c) 独立。

多个事件的相互独立

设 ({E_i}_{iI}) 为一族事件。若对 (I) 的任意非空有限子集 (J)，都有

[ P({jJ}E_j)={jJ}P(E_j), ]

则称这族事件相互独立。

注意：两两独立不一定推出相互独立。

本章小结

  • 概率论用概率空间 ((,,P)) 对随机试验建模。
  • 事件是样本空间中的可测子集，概率测度满足非负性、规范性和可数可加性。
  • 古典概型中，概率计算归结为计数：

[ P(E)=. ]

  • 条件概率描述“已知某事件发生后”的概率更新：

[ P(EF)=. ]

  • 乘法公式、全概率公式与 Bayes 定理是条件概率计算的核心工具。
  • 独立性表示一个事件的发生不会改变另一个事件的概率。
